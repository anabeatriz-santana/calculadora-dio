# ⚔️ Desafio DIO - Calculadora de Partidas Rankeadas

Projeto desenvolvido durante os estudos de lógica de programação na **DIO** (Digital Innovation One).

## 📋 Sobre o desafio

O objetivo é criar uma **função** que recebe a quantidade de vitórias e derrotas de um jogador, calcula o saldo de partidas rankeadas (vitórias - derrotas) e retorna o nível correspondente, de acordo com as faixas abaixo:

| Vitórias | Nível |
|---|---|
| Menor que 10 | Ferro |
| 11 a 20 | Bronze |
| 21 a 50 | Prata |
| 51 a 80 | Ouro |
| 81 a 90 | Diamante |
| 91 a 100 | Lendário |
| Maior ou igual a 101 | Imortal |

## 💻 Conceitos utilizados

- Variáveis
- Operadores
- Laços de repetição
- Estruturas de decisão (if / else if)
- Funções

## 🚀 Como executar

1. Clone este repositório
2. Abra o arquivo `index.js`
3. No terminal, rode:
```
node index.js
```
